# MERN + Tailwind Starter

This workspace contains a minimal MERN scaffold:

- `backend` — Express + Mongoose API server
- `frontend` — Vite + React app with Tailwind CSS

Quick start:

1. Backend:

```powershell
cd backend
npm install
cp .env.example .env        # set MONGO_URI and PORT
npm start
```

2. Frontend:

```powershell
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173` after starting the frontend.
